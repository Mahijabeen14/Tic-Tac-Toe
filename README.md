# 🍭 Candy Land Tic-Tac-Toe 🎮

Welcome to a sweet and playful version of the classic Tic-Tac-Toe game, designed with a *Candy Land theme* and built using *HTML, CSS, and JavaScript*.  
This playful project brings a candy-coated twist to the classic game, perfect for anyone who loves vibrant visuals and fun interaction. 🍬🍭

---

## ✨ Features

- 🍬 Candy emojis instead of boring X and O  
- 🍭 Neon buttons with soft pastel styling  
- 🎉 Pop-up win/draw message with animations  
- 🔁 New Game & Reset functionality  
- 💻 Responsive design – works on desktops and tablets  

---

## 🛠 Built With

- *HTML5*
- *CSS3* (custom animations and styling)
- *JavaScript* (game logic + DOM handling)

---

## 🎮 How to Play

1. Click on a box to place your candy.
2. Players take turns placing 🍭 and 🍬.
3. First to align 3 candies in a row/column/diagonal wins!
4. Use *Reset* or *New Game* to play again.
